# Rental-Vehicles
This program is a simulation of processing a vehicle in a rental agency 
Vehicle Rental Agency (Ongoing progress):
In this program should be capable of maintaining vehicle rental reservations for a vehicle rental agency that rents cars, SUVs and minivans. 
	
There are Car, SUV, and Minivan classes that was child class to the parent class Vehicle. Then a making a class linked list for Vehicle to be stored the all three vehicle. Next using the Vehicle class for making reservation for each vehicle named ReservationDetails and the time period for how long it would be rented. There also was also a linked list class named Vehiclerates for store current rates for each type of vehicle. 
	
After making all the classes, it will then be store in a string array get all rates, vehicles, and transaction into a class named SystemInterface. The idea is use to have the program ask who was using the system. If the input was employee, it would use the EmployeeGUI and the manger would use the ManagerUI.
